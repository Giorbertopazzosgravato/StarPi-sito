# sito_starpi
Questo e' il nuovo mitico sito di StarPi

## Project Setup
``` bash
git clone https://github.com/Giorbertopazzosgravato/StarPi-sito
```

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
